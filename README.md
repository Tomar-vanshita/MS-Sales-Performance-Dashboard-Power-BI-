# MS Sales & Performance Dashboard (Power BI)

## Project Overview
This project focuses on analyzing **sales and performance data** using Power BI to monitor key business metrics and identify trends.

The dashboard is designed to provide a clear, high-level view of performance while allowing users to drill down into detailed insights that support data-driven decision-making.

---

## Why This Project?
I created this project to practice building **executive-style Power BI dashboards** that focus on clarity, KPIs, and business usability.

The goal was to design a dashboard that stakeholders can quickly understand and use for performance tracking and strategic planning.

---

## Business Objectives
- Track overall sales and revenue performance
- Monitor key performance indicators (KPIs)
- Identify trends and patterns over time
- Support management and business decision-making
- Provide a clear performance overview through dashboards

---

## Tools & Technologies
- **Power BI**
  - Data modeling and relationships
  - DAX measures
  - KPI cards and charts
  - Interactive slicers and filters
- **Excel / CSV** (data source)

## Dataset Information
